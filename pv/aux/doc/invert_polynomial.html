  function invert_polynomial.m

  Author


Roman Firment

  Description


Invert polynomial coefficients.

  Input parameters


p ... vector of polynomial coefficients [a_1 ... a_n]

  Output parameters


pi ... if n is odd then it returns
[ a_1 -a_2 a_3 .. -a_(n-1) a_n]
else [ -a_1 a_2 -a_3 .. -a_(n-1) a_n]

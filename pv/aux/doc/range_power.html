  function range_power.m

  Author


Roman Firment

  Description


The range of natural power over interval ix (as a function).

To achieve:

[-8,1] = range_power([-2,1],3) != [-2,1]*[-2,1]*[-2,1]=[-8,4]

  Input parameters


ix ... interval
n ... natural number

  Output parameters


iy ... the range of n-th power function over ix
